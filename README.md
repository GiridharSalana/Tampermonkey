# Tampermonkey Scripts 🚀

A collection of modern, feature-rich Tampermonkey userscripts that enhance your web browsing experience with beautiful UIs and powerful functionality.

## 🎯 About

This repository contains custom userscripts designed to improve various websites with:
- 🎨 Modern glassmorphism UIs
- ⚡ Performance optimizations
- 🔧 Bug fixes for broken site features
- ✨ Enhanced user experiences
- 🎭 Smooth animations and transitions

## 📦 Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Browse the scripts in this repository
3. Click on the script file you want
4. Click "Raw" to view the source
5. Tampermonkey will automatically detect it and ask to install
6. Enjoy your enhanced browsing experience!

## 🛠️ Features

Scripts in this repository may include:

- And much more!
